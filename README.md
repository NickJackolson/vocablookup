# vocablookup

Rofi -dmenu style English vocabulary look up from https://www.vocabulary.com/
Opens the web page in default browser.

# Installation

Just make the script executable and add it to your path
```bash
chmod +x vocablookup
cp vocablookup /path/to/your/$PATH
```

Optionally you can add a word list for suggestions, vocablookup looks
for it in ~/.config/vocablookup/dict file by default.
